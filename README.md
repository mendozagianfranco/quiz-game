# Quiz Game
Lo scopo è realizzare un’app che permetta all’utente di rispondere a una serie di domande e ottenere un punteggio finale.
Il progetto è pensato per mettere in pratica competenze di **React**, **React Router**, **Tailwind CSS** e **gestione dello stato**.

## Tecnologie utilizzate

- React
- Vite 
- React Router DOM 
- Tailwind CSS

##  Installazione e avvio

1. Clona il repository:
```bash
https://github.com/mendozagianfranco/quiz-game.git
```
2. Entra nella cartella:
```bash
cd quiz-game
```
3. Installa le dipendenze:
```bash
npm install
```
4. Avvia il server di sviluppo:
```bash
npm run dev
```
5. Apri nel browser
